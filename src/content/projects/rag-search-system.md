En el marco de la investigación y desarrollo de IA para clientes enterprise, desarrollé soluciones de recuperación de información utilizando el patrón Retrieval-Augmented Generation (RAG).

El sistema utiliza Azure AI Search (Foundry) y la base de datos vectorial Qdrant para indexar y consultar grandes volúmenes de datos corporativos. Las consultas son procesadas por Modelos de Lenguaje Grandes (LLMs) configurados para entregar respuestas precisas basadas exclusivamente en los documentos recuperados. Esto permite al usuario final interactuar en lenguaje natural con su propia base de conocimiento corporativa de manera segura.
